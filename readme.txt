LINGUAGEM: C# para backend (.NET 9) com ASP.NET, SQLite via P/Invoke e SHA-256 para hash de senhas.
O SQLiteHelper é um wrapper que permite chamar as funções da biblioteca SQLite em C a partir do C#, além de gerenciar a conexão com o banco de dados.

ANTES DE RODAR O PROJETO:

SDK 9.0: https://dotnet.microsoft.com/download/dotnet/9
Baixar o SQLite x64: https://www.sqlite.org/download.html e colocar os arquivos na pasta principal do projeto (caso ainda não estejam presentes)
Executar no Git Bash:
./run.sh

http://localhost:5000
